## Installing NGINX
- Now that the VM is setup we need to install NGINX
- Use the SSH command found in vm-setup.md to access the VM
- While in the VM use the command "sudo apt update" this will update the apt repositories
- Next use the command "sudo apt install nginx-full" this will install NGINX onto the VM

## Testing the Web Server 
- In the Azure portal, under "servername-ip" the public IP address can be found
- Using a search engine enter the IP address found
- You will see a "Welcome to NGINX" page verifying eveything is working correctly
